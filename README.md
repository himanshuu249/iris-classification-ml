# Iris Flower Classification using Random Forest

## 📌 Project Overview
This project builds a Machine Learning model to classify Iris flower species using the Random Forest algorithm.  
The workflow includes data preprocessing, train-test splitting, model training, and evaluation.

---

## 📊 Dataset Information
The dataset used is the classic Iris dataset.

Features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

Target Classes:
- Setosa
- Versicolor
- Virginica

---

## ⚙️ Model Used
- Random Forest Classifier (Scikit-learn)
- Train-Test Split: 80% Training, 20% Testing
- Random State: 42
